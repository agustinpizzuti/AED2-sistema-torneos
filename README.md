# Sistema de Gestión de Torneos - AED2

Proyecto académico desarrollado en Java para la materia 
Algoritmos y Estructura de Datos 2.

## Funcionalidades

- Gestión de jugadores
- Gestión de equipos
- Modelado de sucursales mediante grafos
- Detección de sucursales críticas
- Análisis de latencia para torneos

## Tecnologías

- Java
- Programación Orientada a Objetos
- Árboles Binarios de Búsqueda
- Grafos ponderados
