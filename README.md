# 🎮 Sistema de Gestión de Torneos - AED2

Proyecto académico desarrollado en **Java** para la materia **Algoritmos y Estructura de Datos 2**.

El sistema modela la gestión de una empresa de videojuegos que organiza torneos, permitiendo administrar jugadores, equipos y sucursales, aplicando estructuras de datos eficientes y respetando restricciones de complejidad temporal.

---

## Funcionalidades

- Registro y búsqueda eficiente de jugadores
- Gestión de equipos y validación de reglas de negocio
- Asociación de jugadores profesionales a equipos
- Modelado de sucursales y conexiones mediante grafos
- Detección de sucursales críticas (puntos de articulación)
- Selección de sucursales según límites de latencia
- Listados ordenados por distintos criterios

---

## Conceptos y Estructuras Aplicadas

- Árboles Binarios de Búsqueda (ABB)
- Grafos no dirigidos y ponderados
- Recorridos DFS
- Análisis de complejidad (O(log n), O(n), O(k))
- Programación Orientada a Objetos

---

## Tecnologías

- Java
- IntelliJ IDEA
- Implementación manual de estructuras (sin uso de Collections Framework)

---

## Objetivo del Proyecto

Aplicar estructuras de datos avanzadas y algoritmos eficientes respetando restricciones formales de complejidad, simulando un sistema real con reglas de negocio específicas.
